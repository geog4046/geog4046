Lab Assignment 7

Getting started with JavaScript and Leaflet

Create a web map with Leaflet and display a basemap, point, line, and polygon with a popup, using all of the necessary HTML, CSS, and JavaScript. The steps are based on the Leaflet Quick Start Guide. For this assignment, samples of code are given but variable names must be renamed from the generic renameThis to something descriptive. The values (coordinates, popup text, etc.) must also be changed to something you come up with.

A basemap, a point feature (marker), and a polygon feature.

Process

This process assumes you have already set up Atom and Github, and cloned your Github repository to your computer with Atom.

  1. Open Atom, and if it does not automatically open to your Github repository folder, click File > Add Project Folder... and select the folder you cloned your repository to on your computer, e.g., C:\Users\student\github\student4046.github.io.
    Figure 1. If you have created a repository on Github.com, you can clone it to your computer using Atom, and then also use Atom to edit the web page files.
  2. Create a new folder/file called map1/index.html, and for the contents of index.html, paste in the code from the basic HTML skeleton example.
  3. In your map1 folder, create a new JavaScript file map1.js and a CSS file map1.css Figure 2. The tree view in Atom should show the files you just created: index.html, map1.css, and map1.js.

  4. Within your index.html page’s head tags, add a link element to include the CSS file you just created:

    <link rel="stylesheet" href="renameThisFile.css">

    The contents of index.html should now look like this, with the link element added between the head tags:

     <!DOCTYPE html>
 <html lang="en" dir="ltr">
   <head>
     <meta charset="utf-8">
     <title></title>
     <link rel="stylesheet" href="renameThisFile.css">
   </head>
   <body>
        
   </body>
 </html>

    On the line directly above the </body> tag, create a script element to include the JavaScript file you just created:

    <script src="renameThisFile.js"></script>

  5. Also add link and script tags for Leaflet on the lines above your own .css and .js files:

    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.6.0/dist/leaflet.css">
<link rel="stylesheet" href="renameThisFile.css">

    <script src="https://unpkg.com/leaflet@1.6.0/dist/leaflet.js"></script>  
<script src="renameThisFile.js"></script>

    Your HTML, CSS, and JavaScript files are now ready to have code added to them.

  6. Within the body tags, somewhere above the script tags, create a div element where the map will be drawn, using your own value for id:

    <div id="renameThisId"></div>

    Note: The value you choose for renameThisId will be re-used for the id in steps 6 and 7.

  7. In the CSS file, create a rule to give the div a height. Use a different height value for your own map:

    #renameThisRule { height: 180px }

  8. In the JavaScript file, write the first line of JavaScript to create an empty map. Set your own variable name, map id, map center coordinates (in the form [latitude, longitude]), and zoom level:

    var renameThisMap = L.map('renameThis').setView([32.18, -99.14], 4)

    Note: keywords L, map, and setView are specific to Leaflet, while var is a JavaScript keyword used to declare a variable; in this case, a variable to hold your map object. The variable name you choose here for the map object in place of renameThisMap will be used later to add layers to the map.

  9. Add a basemap layer to the map (see basemap providers) using your preferred basemap’s URL and your own map object’s name:

    L.tileLayer('your-basemap-url-here').addTo(renameThisMap)

  10. Mark a point of interest on the map using Leaflet’s marker method. Use your own variable names and coordinates:

    var renameThisMarkerObject = L.marker([30, -90]).addTo(renameThisMap)

    Draw a polygon around an area of interest (use your own coordinates):

    var renameThisPolygonObject = L.polygon([
  [31, -88],
  [35, -88],
  [32, -82]
]).addTo(renameThisMap)

  11. Create popups for your marker and polygon, renaming the variables with the corresponding names you created above:

    renameThisPolygonObject.bindPopup('Your polygon popup text here.')
renameThisMarkerObject.bindPopup('Your marker popup text here.')
  12. Try it: consult the Leaflet documentation (or search online) to add a “polyline” to the map.

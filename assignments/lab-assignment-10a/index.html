Lab Assignment 10a

Bootstrap

Create a website using Bootstrap and add a map using Leaflet. Add other content to the page to get familiar with HTML elements and Bootstrap CSS classes.

Bootstrap is a CSS and JavaScript framework for building modern web pages. It is a library of functions and styles that can save developers from having to write all of their own CSS and JavaScript to make common parts of a website, such as buttons and textboxes, navigation and menus, and layouts that respond to mobile devices.

Process

  1. Create a new folder containing .html, .css, and .js files.
  2. For the initial contents of the HTML and CSS files, see the template created for this class or look for other examples. The templates usually contain an index.html file and a CSS file named for the template, such as starter-template.css.
  3. Examine the HTML code and notice that the head and body already contain link and script tags, respectively. These reference the libraries required by Bootstrap. There is also a reference to starter-template.css.

  4. In index.html, add link and script tags for Leaflet after those for Bootstrap but before your own:

    <!--link element for Bootstrap CSS above-->
<link rel="stylesheet" href="https://unpkg.com/leaflet@1.6.0/dist/leaflet.css">
<!--link element for starter-template.css below-->

    <!--script elements for jquery, popper, and bootstrap above-->
<script src="https://unpkg.com/leaflet@1.6.0/dist/leaflet.js"></script>  
<!--script element for map.js below-->

  5. Create a div element where the map will be drawn:

    <div id="map"></div>

  6. In the CSS file, add a rule specifying at least the height of your map div:

    #map { height: 500px; }
  7. In the JavaScript file, paste the code from a previous assignment to create a map, making sure to update it to use the id you specified for your map div. Test your page for errors. At this point the page should display a map.

  8. Next, try out additional HTML elements to add more content to your page. Not all of these need to be included in the assignment.

    c. Proper indentation
    d. No errors in the console
  2. At least one of each of these page elements:

    b. h1 for a heading
    c. p or span for text
  3. Page elements with Bootstrap classes:

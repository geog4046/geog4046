Leaflet Assignment 1

Getting started with JavaScript and Leaflet

Create a web map with Leaflet and display a basemap, point, line, and polygon with a popup, using all of the necessary HTML, CSS, and JavaScript. The steps are based on the Leaflet Quick Start Guide. For this assignment, samples of code are given but variable names must be renamed from the generic renameThis to something that describes the potential values. The values (coordinates, popup text, etc.) must also be changed to something you come up with.

A basemap, a point feature (marker), and a polygon feature.

Process

  1. In your portfolio repository, create a new folder/file called map1/index.html, with a basic HTML skeleton.
  2. In your map1 folder, create a new JavaScript file webmap1.js and a CSS file webmap1.css
  3. In your index.html page’s head and body, add link and script tags to include your JS and CSS files.
  4. Also add link and script tags for Leaflet on the lines above your own CSS and JS: 
    <link rel="stylesheet" href="https://unpkg.com/leaflet@1.4.0/dist/leaflet.css">

    <script src="https://unpkg.com/leaflet@1.4.0/dist/leaflet.js"></script>

    Your HTML, CSS, and JavaScript files are now ready to have code added to them.

  5. Create a div element where the map will be drawn, using your own value for id: 
    <div id="renameThis"></div>
  6. In the CSS file, create a rule to give the div a height. Use a different height value for your own map: 
    #renameThis { height: 180px }
  7. In the JavaScript file, write the first line of JavaScript to create an empty map. Set your own variable name, map id, map center coordinates (in the form [latitude, longitude]), and zoom level: 
    let renameThis = L.map('renameThis').setView([32.18, -99.14], 4)

    Note: keywords L, map, and setView are specific to Leaflet, while let is a JavaScript keyword used to declare a variable. In this case, let a variable to hold your map object. The name you choose here will be used later to add layers to the map.

  8. Add a basemap layer to the map (see basemap providers) using your preferred basemap’s URL and your own map object’s name: 
    L.tileLayer('https://cartodb-basemaps-{s}.global.ssl.fastly.net/dark_nolabels/{z}/{x}/{y}.png').addTo(renameThis)
  9. Mark a point of interest on the map using Leaflet’s marker method. Use your own variable names and coordinates: 
    let renameThis = L.marker([30, -90]).addTo(renameThis)

    Draw a polygon around an area of interest:

    let renameThis = L.polygon([
  [51.509, -0.08],
  [51.503, -0.06],
  [51.51, -0.047]
]).addTo(renameThis);
  10. Create popups for your marker and polygon, renaming the variables with the corresponding names you created above: 
    renameThis.bindPopup('A polygon')
renameThis.bindPopup('A marker')
  11. Try it: consult the Leaflet documentation (or search online) to add a “polyline” to the map.
